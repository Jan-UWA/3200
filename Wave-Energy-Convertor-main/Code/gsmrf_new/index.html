Introduction

Since the first attempt at harnessing ocean wave energy in 1799, well over a thousand different wave energy converters (WECs) have been invented and tested. However, harnessing wave energy economically remains a challenge. This is partly because of the lack of an established benchmark with which to compare the economic potential of WECs. We have created this database to address this issue. The database is built from existing data in the literature, which are used to calculate various performance metrics to allow comparison of WECs.

The user can interactively upload the power performance curve of a specific WEC along with the associated parameters. The system will calculate a number of different performance metrics, including a new dimensionless metric that pairs capture width with surface area and accounts for the relative cost via a first moment of area computation across a range of wave periods. The user can display the calculated metric scores against the scores of all WECs in the current database.

We hope that this growing database will help place existing and future WECs in perspective with one another in a way that considers not only their ability to capture energy but also their capital cost.
